# Superstore Sales Prediction Dashboard

## Overview
This project analyzes Superstore sales data and predicts future sales using Power BI.

## Features
- Total Sales, Profit & Average Sales KPIs
- Revenue by Category
- Revenue by Region
- Monthly Sales Trends
- Profit by Category
- Actual vs Prediction Sales Correlation
- Top States by Sales
- Interactive Filters (Region, Category, Year)

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Data Visualization

## Dashboard Preview
![Dashboard](dashboard.png)

## Key Insights
- Technology category generated highest revenue.
- West region contributed the highest sales.
- California was the top-performing state.
- Prediction model achieved R² score of 0.78.
